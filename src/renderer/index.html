Getting Started

        
      

      

        
Building a program

        
Drag blocks from the toolbox on the left onto the workspace. Put your logic inside the void setup() block (runs once) and the void loop() block (runs forever) from the Structure category. As you build, the Arduino C++ is generated live in the panel on the right.


        
Toolbox categories

        
    
          
  • Structure — setup() and loop() containers.
    • 
          
  • Arduino I/O — pinMode, digital/analog read & write, map.
    • 
          
  • Time — delay, delayMicroseconds, millis.
    • 
          
  • Serial — Serial.begin/print/println.
    • 
          
  • Logic, Loops, Math, Text — standard programming blocks.
    • 
          
  • Variables, Functions — your own variables and procedures.

The code panel

        
The right panel is a full code editor. It auto-updates from your blocks until you type in it manually — after that, blocks stop overwriting your edits. Click Sync from Blocks to regenerate and discard manual changes. Drag the divider between the panels to resize them.


        
Verify & Upload

        
✓ Verify (Ctrl+R) compiles your sketch with arduino-cli and underlines any errors in the editor, showing the compiler output below. → Upload (Ctrl+U) compiles and flashes it to the board selected in the dropdown — click ⟳ to rescan for connected boards.


        
Files

        
Use the File menu to create, open, and save .ino sketches (Ctrl+N / Ctrl+O / Ctrl+S). Sketches you save embed your blocks so they reopen exactly. Opening a hand-written .ino (made elsewhere) loads its code into the editor for editing.
